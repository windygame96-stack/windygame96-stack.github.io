# 摸鱼之神温迪的游戏仓库

这是一个不需要构建步骤的静态网页版本，直接下载后上传到GitHub仓库即可使用。

## 部署说明

1. 下载此项目的所有文件
2. 上传到你的GitHub仓库
3. 在GitHub仓库设置中启用GitHub Pages
4. 选择合适的分支和目录（通常是根目录）
5. 等待几分钟，你的网页将会在线上可用

## 特点

- 纯静态HTML/CSS/JavaScript实现，无需构建步骤
- 响应式设计，适配各种屏幕尺寸
- 支持深色/浅色主题切换
- 包含4个游戏项目展示
- 平滑的动画和过渡效果
- 通过CDN引入所需资源

## 游戏项目

1. 逃离宏业电子厂
2. 肌肉的诱惑
3. 地铁抢座大作战
4. 电梯ELEV-9

## 自定义

如果你想修改内容，可以直接编辑`index.html`文件：
- 更改标题和描述
- 更新游戏项目信息
- 修改样式和颜色
- 添加或删除游戏卡片
